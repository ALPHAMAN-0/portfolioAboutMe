# Image Placeholders

Please add the following images to this folder:

## Required Images:

1. **workspace.jpg** (1920x1080 recommended)
   - Professional workspace/desk setup image for the home page hero section
   - Should show a clean, modern workspace with computer, keyboard, etc.

2. **portrait.jpg** (800x800 recommended) 
   - Professional portrait photo for the about page
   - Should be a clear headshot or upper body photo

3. **portfolio1.jpg to portfolio6.jpg** (800x600 recommended)
   - Six portfolio project images showcasing your work
   - Can be logos, websites, designs, or other creative work
   - Name them portfolio1.jpg, portfolio2.jpg, etc.

## Image Guidelines:

- Use high-quality images (300 DPI if possible)
- Optimize for web (compress to reduce loading time)
- Maintain professional appearance
- Ensure good contrast for readability

## Alternative Sources:

If you need placeholder images for testing:
- Unsplash.com (free stock photos)
- Pexels.com (free stock photos) 
- Your own photography/design work

The website will work without images, but they greatly enhance the visual appeal.
