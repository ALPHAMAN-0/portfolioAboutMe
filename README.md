# Jason Martin - Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript featuring a dark theme and smooth animations.

## Features

- 🎨 Modern dark theme design
- 📱 Fully responsive layout
- ✨ Smooth animations and transitions
- 🎯 Interactive progress bars
- 🖼️ Portfolio filtering system
- 📞 Contact information display
- 🔄 Smooth page transitions
- 🎪 Parallax effects
- 📧 Social media integration

## Pages

1. **Home** - Hero section with professional workspace image
2. **About** - Personal information and background
3. **Resume** - Skills, experience, and education
4. **Portfolio** - Project showcase with filtering

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript ES6+
- Font Awesome Icons
- Google Fonts (Inter)

## Installation & Setup

1. Clone or download this repository
2. Add your own images to the `images/` folder:
   - `workspace.jpg` - Professional workspace image for home page
   - `portrait.jpg` - Personal portrait for about page
   - `portfolio1.jpg` to `portfolio6.jpg` - Portfolio project images

3. Customize the content in HTML files:
   - Update personal information in `about.html`
   - Modify skills and experience in `resume.html`
   - Add your own portfolio projects in `portfolio.html`

4. Open `index.html` in your web browser

## Customization

### Colors
The website uses a dark theme with these main colors:
- Primary: `#00d4ff` (Cyan blue)
- Background: `#1a1a1a` to `#2d2d2d` (Dark gradient)
- Text: `#ffffff` (White)
- Secondary text: `#cccccc` (Light gray)

### Fonts
- Primary font: Inter (Google Fonts)
- Font weights: 300, 400, 500, 600, 700

### Images
Replace the placeholder images in the `images/` folder with your own:
- Use high-quality images (1920x1080 or higher)
- Optimize images for web (compress to reduce file size)
- Maintain aspect ratios for best results

### Content
1. **Personal Information**: Update name, title, email, and contact details
2. **Skills**: Modify skill levels and add/remove skills in resume.html
3. **Experience**: Add your work experience and education
4. **Portfolio**: Replace with your own projects and descriptions
